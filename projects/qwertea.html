Overview

QwerTEA Cafe is a fictitious internet cafe that sells internet access time as well as drinks such as boba tea and other caffeinated drinks. The pages that were developed in this project are purchase processing forms.

Main Cafe Page

The main cafe page includes a fields where users can fill in their name and specify how many milk tea and internet packages. The "Order Summary" and "Sub Total" sections are updated once the "process order" button is clicked.

Try the Main Cafe Page

Grand Opening Cafe Page

The grand opening cafe page includes form input fields where users can input their first and last name, phone number (in a specific format), and number of orders. The cost per item is also displayed for easier viewing, and the calculated discounts and sub total is updated once the "Process Order" button is clicked.
